## Installation
Add this line to your application's Gemfile:

```ruby
gem "sidekiq-activerecord-shard-middleware"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install sidekiq-activerecord-shard-middleware
```
